LRC to JSON Converter

How to Use

This tool converts .lrc files into a JSON format that's only compatible with Psych Deluxe.

  • Upload an .lrc file or paste its contents into the text area.
  • Click Convert to generate a JSON file.
  • Download the result and place it in your mod's data/ folder.

Note: This won't work with vanilla FNF or other engines. It's designed specifically for Psych Deluxe.
